Socket网络编程

网络给我带来什么好处?:  多人玩游戏  聊天 视频 ...
 可以实现多个计算机相互通信、数据传递和数据共享

多个计算机如何相互通信?
IP地址:  32位 ，分成四组 每组都是八位的二进制数
00001111 101010010 ....  由于不好记忆   又把二进制数转换成十进制数
192.168.1.254
查看ip ?  通过cmd  命名提示符窗口  通过 ipconfig
通过 ping 对方的ip的地址   （会向对方的网络发送4个数据包,如果没有丢失 网络是通畅的）

在使用网络时,没有输入ip访问 某个网址 比如：百度
由于 百度设置了动态ip    ip数字也不好记忆
ip也可以定义多个    所以存在了DNS域名解析服务器
通过网址输入的指定的内容 就可以通过dns服务器解析成对应的ip 地址
总结: 可以通过ip地址 可以定位那台计算机

网络通信协议: 在网络中,实现多个计算机相互通信,设定了相同规则和约定
a.  TCP: 网络传输控制协议
b.  UDP: 用户数据报协议

端口号:  用于区分同一台计算机不同的应用程序(进程)
取值范围: 1~65535    1~1000+  都是系统默认端口
如: http: 默认 80端口     ftp:21端口
    oracle: 1521端口   mysql  3306 端口

总结:只有通过ip + 端口 才可以实现不同的计算机 程序和程序之间的通信

Socket: 在网络上运行的程序，可以实现双向通信中介点。
通过ip地址+端口号 就可以访问到对应scoket对象
sokcet存在 服务端 和 客户端

Socket实现原理:
1. 包括了服务端 server和客户端 client
2. 服务端提供对应端口号, 客户端通过配置ip地址+端口链接服务端
3. 服务端和客户端就是相互发送数据
4. 通过I/O 流  进行通信
5. 关闭资源
