static  修饰符  表示静态的
可以修饰: 属性  方法   代码块
通过static 修饰的属性  表示类的本身(
通过类创建的对象共用同一个静态属性)  也叫静态变量 也叫类变量
通过static 修饰的方法  属于静态方法 或者 类方法
如何调用静态属性和静态方法
1. 类名.静态属性   类名.静态方法
2. 创建对象   对象名.静态属性、静态方法

静态方法里面只能使用 静态变量和静态方法
普通方法可以任意使用

为什么要使用static?
a. 创建多个对象的时候，多个对象共享一个静态属性和方法,所以如果
有一个对象改变了它，那么其他人也会改变
b. 一个类中只有同一个静态属性和方法，在创建多个对象时，每个
对象不用再多余分配空间了  节省空间  因为他们共用同一个

static 修饰  代码块
代码块:
构造代码块 :   创建对象会执行
普通代码块 :   调用方法会执行
静态代码块 :   在类加载(使用了类中的资源就要先加载类)会执行,
               所以永远会第一行执行，而且只会运行一次

继承: 将共同的特征(属性、方法)提取出来,写到一个类中，其他类只要继承
这个类，其他类就有了这个类的共同特征  提高代码的重用性
使用extends关键词    java中一般类 只支持单继承(一个类只有一个直接父类)
但是可以支持间接继承   A==>B==>C==>D
子类可以使用父类非私有资源， 父类不可以使用子类的内容

abstract 修饰符: 修饰类 和方法  表示 抽象类和抽象方法
抽象特点:
1. 抽象方法  只有方法的声明 没有实现
   public abstract void test();
2. 有抽象方法的类  类必须是抽象类
3. 抽象类  可以没有抽象方法
4. 抽象类一般情况下不能被实例化(不能new)
5. 子类继承的父类是抽象类, 如果父类有抽象方法，
子类必须实现父类所有抽象方法, 否则 子类也是抽象类
