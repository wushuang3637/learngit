2.java特点：
   java 面向对象为核心  高并发   多线程   跨平台  强类型高级编程语言
   面向对象:   已类和对象的形式进行编程，更加符合人的思维 
               适合大型项目的研发
    高并发  多线程: 可以同时处理多个事情   人数非常高
    跨平台：  跨越操作系统 win7  8  10  xp  linux  unix ......
    js是弱类型   var s=10   s="123"
   java 是强类型     声明数据类型 = 赋值必须是符合类型的值

7. 编程步骤
    1.   编写源程序   abc...        abc.java
    2.   通过java环境  编译源程序  生成一个二进制文件
             .class   也叫字节码文件
          通过是window  cmd 命令提示符窗口     
          dir:  展示目录
          cd：进入目录
       编程程序之前 保证安装好了 java环境（jdk+环境变量）
       javac 文件名.java   ===> 文件名.class
    3. 运行程序
       java 文件名

8.  java基础语法:
    public  class  类名{

                         数字不能开头   Test1  1Test     test  Test
                         类名必须跟文件名是一致的
        公开的    静态的   没有返回值类型的      方法名(参数)
                                                //如果是程序的入口
                                                方法名：main
                                                参数: String[]
        public   static   void                   main(){

        }
    }

9.  java定义变量（强类型）
 变量:用于保存数据并且处理的方式  类似于定义js变量
      var s=100;  var str="abc";
 java:
 数据类型  变量名=符合数据类型的值
 java数据类型有哪些?
  整型变量4种
  byte  字节     占1个字节   占了8位   00000000~11111111   -2^7--2^7-1   -128~127
  short 短       占2个字节   占了16位       -2^15 --2^15-1
  int   常用的   占4个字节   占了32位      -2^31--2^31-1
  long  长       占8个字节   占了64位      -2^63--2^63-1

  浮点类型2种
  float    单精度    占4个字节
  double   双精度    占8个字节   常用的

  boolean   布尔类型   true/false
  char      字符类型   'a'  '1'  '+'  只能存储一个字符
               有三种赋值方式

  以上 是java八种基本类型

  String 常用的  表示字符串  它不是基本数据类型
         底层实现是引用类型
