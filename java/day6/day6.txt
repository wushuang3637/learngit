面向对象:
类:   具有相同特征事物  抽象出来的概念
对象: 现实世界中客观存在事物
类是对象的模板  , 对象是类的实例

面向对象基本特性:
抽象:
封装:  为了保证属性安全性，让其他类不能直接访问
       a.属性私有化   private
       b.setXXX() 设置值  getXXX() 获取值

如何定义方法：
访问修饰符  返回值类型  方法名(参数列表){
}
访问修饰符
      private    本类
      默认的     本类   同包
      protected  本类    同包  子类
      public     本类    同包  子类   其他

参数: 形参:  方法定义写的参数   (int a,int b)
      实参:  方法调用时传递实际的值   (100,200)
      实参参数传入时要跟形参定义的规则一样  类型和顺序

变量:
    成员变量和局部变量区别
    1.初值不同  成员有初值  局部没有初值 所以 没有赋值不可以使用
    整型变量  0
    浮点型   0.0
    char     ''
    boolean   false
    String,数组,对象 等 特殊类型都是null
    2. 使用范围不同  局部在哪里定义只能在哪里使用
    成员 整个类都可以使用
    3. 优先级不同   局部优先于成员

构造方法: 特殊的方法  关键词 construtor
语法规则:  1. 没有返回值  void 都不能写
           2. 方法名  必须跟类名一致
访问修饰符  方法名(参数){
}
作用: a. 用于创建对象    b. 给属性赋初始值


this.属性     调用当前类的属性
this.方法名() 调用当前类的普通方法
this()  调用当前类的无参构造
this(参数) 调用当前类的有参构造
this() 和this(参数)  必须写在构造方法中 而且必须是第一行


构造方法可以定义很多个  由于java类中支持方法重载的问题
方法重载 overload
方法名 必须相同  参数列表必须不一致(个数、类型、顺序)
跟返回值无关

public Dog(){}
public Dog(int a){}
public Dog(int a ,int b){}
public Dog(int b,int a){}   //跟上面的参数一致了 不属于重载
public Dog(int b,String a){}
public Dog(String b,int a){}
普通方法也支持 方法重载  用法和构造的重载是一致的
